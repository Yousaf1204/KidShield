üìñ Project Setup Guide

‚úÖ Step 1: Clone Repository

‚úÖ Step 2: Firebase Setup

1. Create a new project in Firebase Console.
2. Enable Email/Password Authentication.
3. Set Realtime Database rules:

{
  "rules": {
    "users": {
      "$uid": {
        ".read": "auth != null && auth.uid == $uid",
        ".write": "auth != null && auth.uid == $uid"
      }
    }
  }
}

4. Register Android app with package name com.yousafdev.KidShield 5. Place google-services.json in KidShield/app/
